Role: Senior Frontend Engineer (React)
Company: FinTech Innovators Inc.

About the Role:
We are looking for a Senior Frontend Engineer to lead the development of our highly scalable financial dashboard. You will work closely with cross-functional teams to architect robust web applications and drive our UI/UX modernization efforts.

Key Responsibilities:
* Architect, build, and maintain highly scalable and responsive web applications using React.js and modern state management tools (e.g., Redux, Context API).
* Obsess over web performance; strictly monitor and optimize Core Web Vitals (LCP, CLS, FID) to ensure sub-second page load times.
* Collaborate closely with backend engineering teams to design and integrate RESTful APIs and microservices.
* Champion code quality by establishing strict TDD (Test-Driven Development) practices and conducting rigorous code reviews.
* Mentor junior developers and drive the adoption of frontend best practices across the engineering org.

Required Qualifications:
* 4+ years of professional experience building complex Single Page Applications (SPAs).
* Deep expertise in JavaScript (ES6+), React, and CSS architecture.
* Proven track record of reducing technical debt and resolving complex frontend bug reports from enterprise customers.
* Familiarity with backend integrations (Node.js or Python experience is a huge plus).
