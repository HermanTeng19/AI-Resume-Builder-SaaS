# AI Resume Builder SaaS - Lowest Cost MVP Execution Plan

## 1. 核心战略定位
**目标**: 构建一个低成本、高转化、SEO 友好的在线简历润色与生成平台。
**盈利模式**: 初期的 Google AdSense 广告收入 + 后期的增值会员 (SaaS)。
**核心受众**: 科技互联网从业者、程序员、数据工程师（对 ATS 敏感且具备基本技术认知）。

## 2. 三阶段极简执行路线图

### 阶段一：纯静态单页应用 MVP (0 服务器成本)
抛弃复杂的后端和数据库，利用现代化纯前端架构快速试错。

*   **技术栈**: Vite + React/Vue (或纯 Vanilla JS)
*   **托管平台**: Vercel 或 Cloudflare Pages (永久免费，CDN 全球加速)
*   **AI 算力成本转移 (Hybrid Key Strategy)**:
    *   **免费“诱饵”体验**: 利用 Vercel Edge Functions 封装内置 API Key，配合 IP 限流，允许每个用户每天免费生成 2-3 次。降低小白用户的尝试门槛。
    *   **BYOK (Bring Your Own Key) 模式**: 用户如果想无限次微调简历，需输入自己的 OpenAI/Gemini API Key（Key 仅保存在浏览器 LocalStorage 中，绝对安全）。将最大的算力成本完美转嫁给用户。

### 阶段二：流量变现与 SEO 护城河
在跑通核心功能后，重点解决流量来源和广告变现。

*   **精准的广告点位**:
    *   **"Loading..." 页面广告**: AI 生成简历通常需要 10-30 秒，此时用户注意力高度集中且无法离开，是放置 AdSense 广告转化率最高的黄金位置。
    *   **下载/导出页面侧边栏**: 布局高点击率的职场/培训类相关广告。
*   **垂直领域 SEO 增长飞轮**:
    *   放弃繁重的泛科技聚合网路线。
    *   在应用同域名下搭建 `/blog` 目录，利用 AI 批量撰写职场垂类干货（如：“2026 北美数据工程师 ATS 简历优化指南”、“如何用 STAR 法则重构你的简历”）。通过高维度的长尾关键词拦截搜索引擎流量，直接将读者转化为工具用户。

### 阶段三：全面 SaaS 化 (市场验证通过后)
当 AdSense 收入跑正且每日拥有稳定活跃用户后，开始投入研发成本将其转为标准 SaaS 平台。

*   **技术升级**: 引入 Firebase (Auth 身份验证 + Firestore 数据存储) + Stripe (支付系统)。
*   **Premium 付费会员权益 ($9.9/月 或一次性买断)**:
    *   免除所有 AdSense 广告。
    *   免自带 API Key，提供开箱即用的高级 AI 模型服务。
    *   解锁专属的 10+ 款顶级设计师级 Web/PDF 简历模板。
    *   提供长期的云端简历版本管理和在线域名托管。

## 3. 下一步启动建议 (Next Steps for New Project)
当决定开启这个新项目时，无需从零开始，可以直接基于当前的 `onlineProfile-builder` 经验进行架构演进：
1.  **构建编辑器 UI**: 设计分栏布局（左侧数据录入/PDF解析，右侧实时 HTML 预览）。
2.  **构建 Prompt Pipeline**: 编写结构化的系统提示词，强制 AI 输出严格匹配你已设计好的 UI 组件结构的 JSON 数据。
3.  **接入大模型**: 在前端直接集成相关大模型 SDK 进行流式 (Streaming) 响应输出。
